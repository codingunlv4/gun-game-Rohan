SETUP IN GODOT
  1. Download the model from Sketchfab as GLB.
  2. Save it to: assets/models/golem/sketchfab/golem.glb
  3. Reopen the project so Godot imports the file.

Until golem.glb is added, the game uses a built-in stone golem placeholder.
The slam attack (wind-up lift, ground pound, AoE damage) works with both models.
